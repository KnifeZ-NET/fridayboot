# FridayBoot

#### 介绍
一个基于Springboot开发的后台管理框架

#### 软件架构
- Java 17
- Springboot 2.x
- Spring Security
- Mybatis-plus
- Mybatis-plus-generator
- MySql
- hutool
- Swagger+Apifox
